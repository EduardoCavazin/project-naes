Visão Geral do Projeto

O Assistente Financeiro é uma aplicação web desenvolvida com Django que permite o gerenciamento completo das suas finanças pessoais. Com uma interface moderna e intuitiva, o sistema oferece ferramentas para controle de despesas, cheques, contas bancárias e categorias financeiras, além de dashboards visuais para acompanhamento dos seus gastos.

Principais Funcionalidades

  • Gerenciamento de despesas com suporte a parcelamento
  • Controle de cheques com status (pendente, compensado, cancelado)
  • Cadastro e gerenciamento de métodos de pagamento
  • Organização de despesas por categorias personalizáveis
  • Administração de múltiplas contas financeiras
  • Dashboard com resumo financeiro e gráficos
  • Visualização de gastos mensais e distribuição por categoria
  • Design responsivo para acesso em qualquer dispositivo
